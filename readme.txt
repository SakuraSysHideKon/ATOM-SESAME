■ ATOM Lite SESAMEスマートロック操作 ArukasSmartKeys

■ 更新履歴
2023/06/06 新規作成

■ 概要
M5Stack ATOM Liteを使って、セサミスマートロックを操作します。

■ 配布フォルダ構成/サンプル説明

ATOM Lite Smart Lock
 |
 +- ArukasSmartKeys - ArukasSmartKeys.ino　　　　ATOM Liteからセサミスマートロックを操作するスケッチ。
 |
 +- GetChipInfomation - GetChipInfomation.ino　　ATOM LiteデバイスのMACアドレス等をシリアルモニタに表示するスケッチ。
 |
 +- ATOM Lite SESAMEスマートロック操作 ArukasSmartKeys.txt　このドキュメント

■ 開発環境
Arduino IDE バージョン：2.1.0

■ 使用方法
スケッチArukasSmartKeys.inoのAPI KEY・UUID・シークレットキー等を設定し、Arduino IDEに必要なライブラリをインストールし、スケッチをATOM Liteに書き込む。
WiFi設定はスマホアプリ「ESP TOUCH」にておこなう。
